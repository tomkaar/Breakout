    Instructions

  • Under levels you can choose which level you want to edit.
    • Dropdown: Choose which level you want to edit
    • Clear: Clear the level, remove all bricks and blocks
    • Remove: Remove currently selected level
    • Create Before: Create new level before currently selected level
    • Create After: Create new level after currently selected level
  • Under options you can choose what will happend when you click on a cell in the table.
    • Brick: Place Brick, will use input from color and lives
    • Remove: Remove/ clear already placed blocks
    • Color: Change the color
    • Lives: How many lives the block will have, how many time the ball has to hit it before breaking
  • Output: Here you can see your finished level
    • Click on Download to download the level as a .js file, replace the 'levels.js' file in the /game/ folder
      Make sure you use the same name
      After replacing the file, update the page to see the changes in game
